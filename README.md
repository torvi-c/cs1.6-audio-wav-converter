# cs1.6-audio-wav-converter

*How to use: 
Take your wav file/files and drag and drop them onto the .EXE file. RECOMMENDED TO PUT THE EXE FILE INTO A SEPERATE FOLDER TO AVOID CLUTTER. There is not limit to how many files you can drag and drop in one go so go all out.*
